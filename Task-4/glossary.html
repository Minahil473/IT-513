Web Development Glossary

HTML
HyperText Markup Language, the structure of web pages.
CSS
Cascading Style Sheets, used for styling web pages.
JavaScript
A programming language for web interactivity.
Frontend
The visible part of a website the user interacts with.
Backend
The server-side logic that powers a website.
API
Application Programming Interface, enables communication between software.
SEO
Search Engine Optimization, improving site visibility in search engines.
Responsive Design
Design approach that makes websites adaptable to all screen sizes.
